#### 基础配置

黑果小兵快速镜像：Monterey 12.6.1.dmg

macOS：Monterey 12.6.1

OpenCore:0.8.6

OpenCoreconfigurtor：2.64.0.1

该OC支持升级mac OS 13



#### 电脑配置

主板：B450m mortar max

CPU：AMD 3500X

SSD：SATA

显卡：蓝宝石MAD590白金版

声卡：ALC892

网卡：主板自带+AX200



#### 功能受限

1. 虚拟化无法开启（Docker等虚拟机无法使用，AMD CPU引起）
2. adobe相关软件无法使用，github有修复工具，但不一定好用，自行解决（AMD CPU引起）
3. 隔空投送无法使用（仅部分博通免驱网卡全支持隔空）
4. AX200网卡网络速度不算很快
